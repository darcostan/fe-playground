⚡ Signals Deep Dive

Explore Angular's reactive primitives: signal(), computed(), effect(), and linkedSignal().

1. signal() + computed()

Create reactive state with signal() and derive values with computed().

fullName (computed): {{ fullName() }} 
firstName = signal('Angular');
lastName = signal('Student');
fullName = computed(() => `${this.firstName()} ${this.lastName()}`);

3. linkedSignal()

linkedSignal() creates a writable signal whose value resets when its source changes.

selected(): {{ selected() }} (resets to first option when options list changes) 
options = signal(['A', 'B', 'C']);
// Resets to options()[0] whenever options changes
selected = linkedSignal(() => this.options()[0]);

5. Signal with Object (Immutable Updates)

When signals hold objects, use update() with spread to create new references.

name: {{ user().name }}
age: {{ user().age }}
role: {{ user().role }}
user = signal({ name: 'Alice', age: 25 });

// Immutable update with spread operator
incrementAge() {
  this.user.update(u => ({ ...u, age: u.age + 1 }));
}
